# 列表

列表是一个有序的，可修改的，元素以逗号分割，以中括号包围的序列。

# 列表的定义

  1. []
  2. list
  3. range
  4. xrange

print [1,3,[3,4]]
print list('abcd')
print range(2,10,2)
print list(xrange(10))

# 元组和字符串的区别

  • 元组和字符串都是有序的，不可被修改的序列
  • 元组的元素可以是任何元素，但字符串的元素只能是字符
  • 元组的元素长度可以任意，而字符串的元素长度只能为1
