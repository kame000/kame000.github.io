                        1.8 字典和集合

                        刘文奇 2018-4-12 Python

                        # 一、 字典

                        # 1 映射类型数据结构介绍

                        • 在字典存放的每个元素都是以一对儿键值对。
                        • 在 Python 中 字典通常被称为 dict，键 称为 key，值称为 value
                        • 字典中不可以存在相同的 key，但是 value 可以。

                        # 4. 哪些数据可以作为字典的 key

                        key 通常是 字符串

                        它可以是 Python 中任意不可变类型

                        比如：

                        • 布尔型 True 1 False 0
                        • 整型 100 200
                        • 浮点型 1.0 3.415
                        • 元组 （1,） （1, 2, 3）
                        • 字符串 'host_name'

                        关于字典的 key

                        在 Python 内部用一个哈希表来维护字典中的 key 到 value 的映射关系。

                        所以 key 必须是可哈希的。

                        判断一个对象是否可哈希，可以使用 hash() 函数

                        返回一个整数，就是可哈希，反之会抛出 TypeError 异常

                        hash(1)
hash('name')
hash((1,2,3))

# 列表是可变的数据类型，所有不可以被 hash，当然也就不能作为 字典的 key
hash([1,2,3])

                        # 9. 使用 = 修改或更新字典

                        可以使用 等号 对字典的 key 进行直接赋值操作。

                        In [50]: d5 = {}

In [51]: d5['a'] = 1

Out[53]: {'a': 1, 'b': 2}

In [54]: d5['li'] = [1,3,5]

In [55]: d5

                        # 作业练习

                        # 员工查询系统开发

                        # 需求

                        1. 用户通过输入员工姓名的拼音全拼 , 查询员工信息。

                        2. 假如系统中有此员工，则询问用户想要查询什么信息，比如 地址 、电话等。 并提示用户查询的信息类型如 提示：请输入查询的信息类型 [ p 查询电话，addr 查询地址 ]

                        3. 输入正确后，返回对应的信息

                        4. 假如不存在，则提示查询的员工信息未录入系统。

                        5. 系统一旦运行，保持运行状态，用户输入 b 返回到到上一层，输入 q 退出系统。
