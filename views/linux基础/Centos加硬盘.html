    刘文奇 1/11/2017 Centos硬盘

    # Centos 增加硬盘

    # 环境

    在正常运行CentOS 7.1的机器上添加一块80G的新硬盘.

    # 系统

    # cat /etc/redhat-release

# uname -a

    • 挂载新硬盘，并添加到 /etc/fstab自动挂载

    mount /dev/sdb1 /newdisk

      # Created by anaconda on Wed Apr 15 18:50:24 2015
